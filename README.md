Movie Ratings Data Analysis (Python)

This project performs Exploratory Data Analysis (EDA) on the IMDb Top 1000 Movies dataset to uncover insights into audience preferences, popular genres, and the factors influencing ratings.

Tools Used
Python
Pandas, NumPy
Matplotlib, Seaborn

Highlights
Cleaned and transformed raw data for analysis  
Analyzed genre-wise IMDb ratings and Meta scores  
Visualized runtime and rating relationships  
Identified top-performing directors and genres  

Outcome
Derived actionable insights into audience behavior and movie success metrics using Python-based data analysis techniques.
